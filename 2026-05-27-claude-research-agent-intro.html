核心工具

不需要写复杂代码。技术栈非常清晰，每个组件都有免费额度。

🤖

Claude Code

主要工作台。用来创建和运行 Agent，内置 sub-agent 功能可以并行处理多个任务。Claude Max 订阅或 API 访问均可。

🔍

Tavily API / Perplexity API

负责搜索和抓取网页内容。Tavily 有每月 1000 次免费额度，Perplexity 需要 $5 Credits 起充。两个选一个即可。

🌐

Firecrawl / Jina.ai

网页内容提取。Firecrawl 每月 500 页免费；Jina.ai 完全免费但速度稍慢。用于将网页转成结构化文本。

⚙️

MCP Servers

Model Context Protocol。桥接 AI Agent 和外部工具（搜索、爬虫、API）的标准协议。Claude Code 原生支持。

Agent 的 prompt 怎么写

核心在于告诉 Agent：你是谁、做什么、输出什么格式。下面是一个经过验证的 prompt 结构。

research-agent-prompt.md
# research-agent.md SYSTEM: "You are a research assistant that creates concise, accurate morning briefs. You synthesize information from multiple sources and identify the key insights. Format your output as: Top 3 Headlines (one sentence each), 3 Key Bullets per headline, 2 Suggested Actions, and relevant links." USER: "Morning briefing for [DATE]. Sources: [SOURCE_LIST]. Recent content: [CONTENT_SNIPPETS]. Focus areas: [TOPICS]. Keep it under 500 words." # Brand Context Profile (可选) # 告诉 Agent 你的背景和偏好，它会据此调整内容筛选 "I'm in the AI/Crypto space, interested in: agent tools, DePIN, Bitcoin ecosystem. I write for a technical audience. Skip generic news, focus on: new tools, data insights, unique angles."
terminal
# 在 Claude Code 中运行 Agent $ claude agents Agent View 已启动 # 输入任务 $ Give me the morning briefing for today Running research agent... Synthesizing 12 sources... Brief generated in 4m 32s

免费额度够用吗
