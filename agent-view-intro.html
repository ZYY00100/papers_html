功能介绍

Agent View 是什么

Claude Code 给了一个专门的界面，叫 Agent View。运行一条命令，你就能同时管所有后台任务。

操作指南

怎么用

上手很简单，记住这几个核心操作就够了。

1

分发任务

在底部输入你想让 AI 干的事，回车，它就启动一个新的独立 session 开始跑。可以同时输入多个任务。

2

查看进展

选中那一行，按空格，弹出 peek 面板，显示它最近在做什么、输出了什么。不用离开当前界面。

3

直接回复

在 peek 面板里直接打字回复，回车发送。整个过程不用离开 Agent View。

4

深入操作

选中那行，按回车或右箭头，钻进那个 session，像正常使用 Claude Code 一样去操作。按左箭头回到 Agent View。

5

整理界面

Ctrl+T 钉任务到顶部、Ctrl+S 切换分组方式、Ctrl+R 给任务改名——让界面始终保持清晰。
